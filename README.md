# 🚇 Московское метро | Статус работы онлайн

Актуальный статус работы Московского метрополитена с учётом текущего времени, закрытых станций и выходов.

## 🌟 Особенности

- **Автоматическое определение статуса** — показывает, работает ли метро сейчас
- **Закрытые станции и выходы** — актуальные данные на 2026 год
- **Расписание** — первые и последние поезда, интервалы движения
- **Все линии метро** — 15 линий с цветовой маркировкой
- **Адаптивный дизайн** — удобно смотреть с телефона
- **Без сервера** — работает полностью на клиентской стороне

## 🚀 Как использовать

Просто откройте `index.html` в браузере или перейдите по ссылке на GitHub Pages.

## 📊 Источники данных

- Официальный сайт Московского метрополитена
- Департамент транспорта Москвы
- Актуальные графики ремонтов на 2026 год

## 📱 Совместимость

Работает во всех современных браузерах и на мобильных устройствах.

## 🔄 Обновление данных

Данные обновляются автоматически при загрузке страницы. Статус работы пересчитывается каждую минуту.
